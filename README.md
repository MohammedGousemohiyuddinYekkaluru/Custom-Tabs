# Custom-Tabs

A simple HTML/CSS/JavaScript project that implements custom tabbed navigation.

## Files
- `index.html` – contains the tab structure and content.  
- `style.css` – styles the tabs and their content.  
- `script.js` – handles tab switching logic.

## Usage
1. Clone the repository:  
   ```bash
   git clone https://github.com/MohammedGousemohiyuddinYekkaluru/Custom-Tabs.git 
2. Open index.html in your browser.

3. Click on the tabs to switch content.
